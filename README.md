# Pokemendel Journey

A local web app for managing Pokemon ROM files.

## Setup

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Run

```bash
source venv/bin/activate
python run.py
```

The app starts at [http://localhost:5000](http://localhost:5000).

## Configuration

Edit `config.yaml` to set ROM paths and add games.
